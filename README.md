# portfolio.ankul.co.in

Professional portfolio for Ankul Choudhry — Senior Engineering Manager.

**Live site:** [portfolio.ankul.co.in](https://portfolio.ankul.co.in)

## Overview

Portfolio showcasing 14+ years of experience building scalable platforms in fintech, insurance, and e-commerce.

### Sections

- **About** — Background, experience, and philosophy
- **Leadership Principles** — Engineering leadership approach
- **System Designs** — Architecture patterns and decisions
- **Case Studies** — Real-world project deep dives

### Related Sites

- [ankul.co.in](https://ankul.co.in) — Main landing page
- [Engineering with Intent](https://ankul.co.in/engineering-with-intent/) — Architecture decisions and operating practices

## Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) with App Router
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Fonts**: [Geist](https://vercel.com/font)
- **Deployment**: GitHub Pages (Static Export)

## Getting Started

### Prerequisites

- Node.js 20+
- npm

### Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build

```bash
npm run build
```

The static output will be in the `out/` directory.

## Deployment

Deployed automatically via GitHub Actions on push to `main` branch.

Custom domain: `portfolio.ankul.co.in`
